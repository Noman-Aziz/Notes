# 10. Expressions + Functions

### Expressions

* Template strings
* Operators (!, -, \*, /, %, >, ==, etc..)
* Conditionals ( cond ? true : false )
* For (\[for o in var.list : o.id])
* Splat (var.list\[\*].id)
* Dynamic blocks
* Constraints (type and version)

***

### Functions

* Numeric
* Date & Time
* String
* Hash & Crypto
* Collection
* IP Network
* Encoding
* Type Conversion
* Filesystem
